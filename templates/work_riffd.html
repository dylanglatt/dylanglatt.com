Riffd

Live riffdlabs.com ↗

Search any song by name and Riffd finds it via Spotify, separates the audio into individual stems, and gives you a full musical breakdown — key, harmony, chord structure, and an interactive mixer so you can hear exactly what each instrument is doing.

The mixer gives you per-stem control over volume, solo, and mute. Strip out the vocals, isolate the bass, or dial in any combination while you play along. The key analysis shows you where the song sits on the circle of fifths, the diatonic chords available in that key, common progressions, and how it relates to neighboring keys.

How it works

Audio comes in via YouTube or file upload. Demucs runs stem separation on cloud GPU, isolating up to six stems with a second melodic pass that splits guitar and piano into lead and accompaniment. Basic Pitch converts each stem into note events; Essentia detects key and BPM directly from the audio signal, surfacing results before separation even finishes.

Chord progressions are sourced from tab databases and scored against diatonic templates to produce roman numeral analysis by section. The mixer — mute, solo, volume, loop, pitch transposition — runs entirely in the browser via Web Audio API, with no server roundtrips. The backend is Flask on Render, with a concurrent job queue so multiple users can run analyses simultaneously.

Why I built it

I play music, and I've never found a good way to learn from the songs I actually listen to. Most theory tools teach abstract concepts in isolation — here's what a major scale is, here's a ii-V-I — but nothing connects that to the real music you care about. I wanted to pull up any song and immediately understand what key it's in, what chords are in play, and why it sounds the way it does.

The stem separation side came from a different frustration: I wanted to play along to songs and actually hear each part on its own — not just see a waveform, but genuinely isolate the bass or the guitar and sit with it. Riffd is what happens when those two things live in the same place.

Stack

Demucs Stem separation — Meta's transformer model on cloud GPU, up to 6 stems with stereo field analysis and a second melodic separation pass
Basic Pitch Note detection — Spotify's pitch detection model converts each stem into note events, separating lead guitar from rhythm chords
Essentia Key + tempo detection directly from the audio signal, results available before stem separation finishes
Web Audio API Browser mixing engine — mute, solo, volume, loop, and pitch transposition with no server roundtrips
Diatonic Template Matching Chord progressions scored against diatonic templates to detect key and produce roman numeral analysis by section
Claude API Generates progression names, key context, and theory-based recommendations from detected chords, tempo, and lyrics
Multi-source Audio YouTube acquisition with automatic fallback across multiple extraction methods; file upload always available
Concurrent Job Queue FIFO queue lets multiple users run analyses simultaneously — waitlisted users see live status and are promoted automatically
Flask + Vanilla JS Lightweight Python backend, zero-dependency frontend. SQLite caching, filesystem job state, deployed on Render with Gunicorn
