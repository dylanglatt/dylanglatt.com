The Problem

The global news stream is massive and mostly noise — hundreds of thousands of events logged every day across every country, topic, and source. Finding what actually matters, in real time, requires infrastructure that until recently only existed inside enterprise and government platforms.

What I Built

I designed and directed the build of a Palantir-inspired event intelligence platform for independent use — same design philosophy, same data sources used at the government and enterprise level, no proprietary infrastructure required.

How I Built It

The system pulls live global event data every 15 minutes and scores each event by severity and relevance. An AI layer reads the underlying news source for anything ambiguous, making a judgment call on whether it’s signal or noise. Events that pass are cross-checked against NASA satellite data for independent ground confirmation.

The interface is modeled on enterprise intelligence platforms — live map, escalation alerts when activity spikes in a region, and real-time situation reports by country. I tested the filtering logic against two years of historical data to refine how the system makes calls.

The Outcome

Argus is live at argusosint.vercel.app. Hundreds of thousands of raw daily events reduced to a clean, AI-filtered, independently corroborated feed — built entirely from public data.

Data Sources

GDELT 2.0 Open public event database updated every 15 minutes — hundreds of thousands of events/day across 100+ countries, sourced from global media
POLECAT CIA/DARPA-funded political event dataset — used to build the historical conflict zone index for satellite corroboration and false positive suppression
NASA FIRMS VIIRS Near-real-time satellite thermal anomaly data — spatially matched to battle and explosion events for independent corroboration
UN OCHA / ReliefWeb Live humanitarian situation reports pulled per country — surfaced in the Country Brief panel for selected regions
CAMEO + Goldstein Scale Standardized event classification and severity scoring — CAMEO codes normalized into actionable categories, Goldstein inverted for impact ranking

Stack

React 19 + Mapbox GL JS Frontend analyst workstation — clustered event map, sortable feed, 6-hour time-series chart, and escalation detection UI
Express.js API layer serving ingested events, corroboration data, and analyst feedback endpoints
Claude Haiku (Anthropic API) Second-tier noise filter — evaluates fetched article excerpts against strict kinetic event criteria, rejecting crime, diplomacy, and non-conflict content
Python data pipeline Scheduled GDELT ingestion, normalization, satellite matching, and 2023–2024 backtesting against POLECAT
Vercel + Vercel Blob Deployment and analyst feedback persistence
